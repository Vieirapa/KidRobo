# KidRobo — frases de fallback do school-demo

Este modo foi criado para a apresentação escolar.

## Objetivo
Quando o STT vier quebrado ou a intenção não ficar clara, o KidRobo não deve cair no Ollama. Em vez disso, ele usa frases locais, curtas, simpáticas e previsíveis.

## Arquivo atual de frases
As frases ficam em:

- `software/app/dialog/school_demo_lines.py`

## Critérios para boas frases
Cada frase deve ser:
- curta
- gentil
- fácil para criança de 5 anos
- segura
- útil para redirecionar a conversa

## Boas direções
As frases devem incentivar coisas como:
- falar mais devagar
- pedir uma curiosidade
- perguntar o nome do robô
- pedir para contar
- pedir brincadeira simples

## Evitar
- frases longas
- humor confuso
- respostas que pareçam erro técnico
- qualquer coisa com tom seco ou frustrado

## Ideia futura
Podemos crescer esse conjunto para dezenas de frases e até separar por categoria:
- fallback curto
- redirecionamento para curiosidade
- convite para brincadeira
- convite para contagem
