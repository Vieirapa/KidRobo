# MVP — módulos para primeira implementação

## Módulo 1 — State machine
Responsável por controlar estados do KidRobo.

## Módulo 2 — Intent classifier
Responsável por detectar perguntas previsíveis.

## Módulo 3 — Safety filter
Responsável por manter respostas curtas e seguras.

## Módulo 4 — Ollama bridge
Responsável por enviar perguntas abertas para o modelo local.

## Módulo 5 — CLI simulator
Responsável por validar o fluxo antes de integrar áudio real.

- STT
