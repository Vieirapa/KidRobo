# Hardware do KidRobo

Esta pasta concentra a documentação de componentes físicos do projeto.

## Objetivo inicial
Montar um protótipo simples com:
- Raspberry Pi 5
- microfone USB
- speaker
- LED RGB
- botão opcional

## Evolução prevista
- display OLED ou olhos em LED
- servo para movimento simples
- Arduino para periféricos e animações
- sensores de toque ou proximidade
